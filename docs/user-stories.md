# コアユーザーストーリー

## オンボーディング
- アプリを見つけたパートナーとして、相手を招待してタスクを共有したい。
- 招待されたパートナーとして、余計な入力なしでワンタップ参加したい。
- カップルとして、賞賛プロンプトの受信頻度を調整し、負担にならないようにしたい。

## 買い物リスト
- 夕食を計画するパートナーとして、メモ付きでアイテムを追加し、必要なものを正確に伝えたい。
- 店頭にいるパートナーとして、チェックした項目が即座に同期され、重複購入を避けたい。
- カップルとして、誰が追加・完了させたかを把握し、役割分担を明確にしたい。

## 家事ボードと賞賛ループ
- 家事を終えたパートナーとして、完了登録と同時に賞賛を送り合ってモチベーションを保ちたい。
- 賞賛を受け取るパートナーとして、簡単にリアクションや返信ができ、感謝を返したい。
- カップルとして、定期的な家事（洗濯・ゴミ出しなど）を追跡し、抜け漏れを防ぎたい。

## 通知とフィードバック
- パートナーとして、その日に相手がしたことをまとめて受け取り、都合の良いタイミングで感謝したい。
- 忙しいパートナーとして、一時的に通知をスヌーズし、過剰な負担を避けたい。
- プロダクトチームとして、賞賛後の感情を軽く収集し、体験を最適化したい。
